## 1. 광고 전략

### 1.1 보상형 광고 배치

| Placement | 상황 | 보상 | 일일 제한 |
|-----------|------|------|----------|
| `craft_speed` | 제작 대기 중 | 제작 시간 50% 단축 | 10회 |
| `craft_double` | 제작 완료 후 | 결과물 2배 | 5회 |
| `recipe_hint` | 레시피 실험 중 | 힌트 1개 | 5회 |
| `cat_energy` | 고양이 피로 회복 | 에너지 풀 | 3회 |
| `daily_bonus` | 일일 보너스 | 2배 | 1회 |
| `shop_bonus` | 판매 시 | 판매액 30% 추가 | 5회 |

### 1.2 전면 광고

- **트리거**: 앱 복귀 시 (30분 이상 이탈 후)
- **쿨다운**: 5분
- **비활성화**: 월정액 구독자

## 4. 재화 밸런스

### 4.1 재화 종류

| 재화 | 용도 | 획득 경로 |
|-----|------|----------|
| 젬 | 프리미엄 구매, 즉시 완료 | IAP, 업적, 광고 |
| 골드 | 재료 구매, 업그레이드 | 판매, 방치, 퀘스트 |
| 재료 | 제작 | 수집, 구매, 방치 |

### 4.2 일일 무료 획득량

- 젬: 30~50
- 골드: 20,000~30,000
- 기본 재료: 100~150개

## 6. 공정성 원칙

- 모든 레시피/고양이 무과금 획득 가능
- 과금은 속도/편의/꾸미기에만 영향
- 고양이 능력치 차이는 미미하게 유지
