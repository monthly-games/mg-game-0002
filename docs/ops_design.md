## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 레시피 밸런스 수집 |
| 3~6주 | 레시피 발견 난이도 조정 |
| 7~10주 | 공방 인테리어 세트 3~4종 추가 |
| 11~12주 | 봄 시즌 테마 준비 |

**핵심 과제:**
- 레시피 발견의 재미 밸런스 조정
- 고양이 캐릭터 애착 형성
- 공방 꾸미기 시스템 안정화

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 봄 테마 레시피 + 고양이 추가 |
| 5개월 | 여름 테마 레시피 + 인테리어 |
| 6개월 | "한정 재료 등장 이벤트" 시작 |
| 7개월 | 가을 테마 레시피 + 고양이 추가 |
| 8개월 | 겨울 테마 준비, 콘테스트 이벤트 |

**핵심 과제:**
- 시즌별 테마 콘텐츠 사이클 정착
- 한정 재료 이벤트 효과 검증
- 유저 커뮤니티 활성화

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 수집 도감 시스템 확장 |
| 10개월 | 업적 시스템 고도화 |
| 11개월 | 스토리라인 확장 |
| 12개월 | 1주년 이벤트, 연말 테마 |

**핵심 과제:**
- 경쟁 요소 최소화 유지
- 수집/도감 콘텐츠 확대
- 장기 유저 리워드 시스템
