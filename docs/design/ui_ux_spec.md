## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 아이템 선택 | 72×72dp | 탭 |
| 아이템 이동 | 셀 → 셀 | 드래그 |
| 머지 실행 | 동일 아이템 위 | 드롭 |
| 아이템 정보 | 아이템 위 | 롱프레스 (500ms) |
| 빈 셀 터치 | 셀 영역 | 탭 (아이템 생성 시) |

### 터치 피드백
- 드래그 시작: 아이템 확대 (1.2x) + 반투명
- 머지 가능: 타겟 셀 녹색 글로우
- 머지 불가: 타겟 셀 빨간 테두리
- 머지 성공: 파티클 + 팝 애니메이션
