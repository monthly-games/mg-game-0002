## 1. 아트 스타일 가이드

### 전체 아트 디렉션
```
Art Style: Cozy pixel art / Low-poly 3D hybrid
Color Palette: Warm, pastel tones with magical accent colors
- Primary: Soft browns, creams, beige (workshop wood/stone)
- Secondary: Pastel greens, light blues (nature materials)
- Accent: Glowing purples, gold shimmer (magical effects)
Mood: Wholesome, relaxing, slightly whimsical
Reference: Stardew Valley meets Unpacking meets Potion Craft
```

### 해상도 및 포맷

- 재료/아이템 스프라이트: 64x64px PNG
- 배경 아트: 1920x1080px PNG
- 캐릭터(고양이): 256x256px 스프라이트시트

## 사용 가이드

### 리소스 생성 워크플로우

1. **우선순위 순서**:
   - 먼저 UI 요소 (버튼, 프레임)
   - 재료/포션 아이콘 (티어 1-2 우선)
   - 고양이 캐릭터 기본 스프라이트
   - 배경 (메인 공방)
   - VFX 효과
   - 음원 (BGM → SFX)

2. **AI 이미지 생성 도구 추천**:
   - Midjourney: 배경, 컨셉 아트
   - DALL-E 3: 아이콘, UI 요소
   - Stable Diffusion: 스프라이트, 픽셀 아트 (LoRA 사용)

3. **픽셀 아트 후처리**:
   - 생성 후 Aseprite나 Pixaki로 정리
   - 색상 팔레트 통일
   - 투명 배경 처리
   - 애니메이션 프레임 조정

4. **음원 생성 도구 추천**:
   - BGM: Suno AI, AIVA, Soundraw
   - SFX: ElevenLabs Sound Effects, 직접 녹음 + Audacity 편집
   - 후처리: Audacity (노멀라이즈, 페이드, 루프 설정)

5. **리소스 최적화**:
   - PNG: TinyPNG로 용량 압축
   - 오디오: OGG Vorbis 포맷 (모바일 최적화)
   - 스프라이트시트: TexturePacker로 아틀라스 생성
