## 3. 상세 BM 설계

### 3.1 꾸미기/인테리어 BM

#### 무료 구조
- 기본 인테리어 세트 3종 무료 제공
- 스토리/업적 진행으로 장식 아이템 획득
- 시즌 이벤트에서 한정 데코 무료 파밍 가능

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 테마 패키지 | 인테리어 세트 (가구 10종+) | $4.99~$9.99 |
| 단품 장식 | 개별 고급 장식 아이템 | $0.99~$2.99 |
| 시즌 한정 세트 | 봄/여름/가을/겨울 테마 | $6.99 |
| 프리미엄 배경 | 공방 배경 변경 | $2.99 |

### 3.2 고양이 수집 BM

#### 무료 구조
- 스토리 진행으로 기본 고양이 5마리 획득
- 일일 방문 고양이 시스템 (랜덤 합류 기회)
- 레시피 마스터 보상으로 특수 고양이 획득

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 고양이 뽑기 | 10연차 1회 | 프리미엄 재화 2,000 |
| 픽업 배너 | 특정 고양이 확률 UP | 동일 |
| 천장 시스템 | 50연차 SR 고양이 확정 | - |
| 고양이 의상 | 개별 코스튬 | $1.99~$3.99 |

### 3.3 제작/성장 BM

#### 무료 구조
- 기본 레시피 50종 무료 발견 가능
- 재료 수집은 방치+간단 퍼즐로 충분
- 공방 기본 업그레이드 무료

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 제작 슬롯 추가 | 동시 제작 +1 | $1.99 |
| 제작 시간 단축 | 즉시 완료 티켓 | 프리미엄 재화 |
| 대성공 부스트 | 성공률 +20% (24시간) | $0.99 |
| 레시피 힌트 | 미발견 레시피 힌트 공개 | 광고 or $0.99 |

### 3.4 방치 수익 BM

#### 무료 구조
- 오프라인 재료 수집 상한: 8시간
- 자동 판매 수익 누적

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 수집 2배 | 광고 시청 시 적용 | 무료 (광고) |
| 수집 3배 패스 | 상시 적용 (30일) | $2.99/월 |
| 오프라인 확장 | 16시간까지 누적 | 패스 포함 |

### 3.5 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 재료 2배, 레시피 힌트, 제작 단축 |

## 6. BM 밸런스 원칙

### P2W 방지선
- 모든 레시피 무과금으로 발견 가능 (시간 차이만 존재)
- 고양이 성능 차이 미미 (외형/개성 중심)
- 인테리어 기능성 버프는 1~2% 수준으로 제한

### 과금 압박 최소화
- 힐링 게임 특성상 긴박한 과금 유도 없음
- 광고 시청도 완전 선택적
- 시간이 지나면 자연스럽게 모든 콘텐츠 접근 가능
